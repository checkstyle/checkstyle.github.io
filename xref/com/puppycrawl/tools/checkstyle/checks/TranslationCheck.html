55  /**
56   * <p>
57   * Ensures the correct translation of code by checking property files for consistency
58   * regarding their keys. Two property files describing one and the same context
59   * are consistent if they contain the same keys. TranslationCheck also can check
60   * an existence of required translations which must exist in project, if
61   * {@code requiredTranslations} option is used.
62   * </p>
63   * <p>
64   * Consider the following properties file in the same directory:
65   * </p>
66   * <pre>
67   * #messages.properties
68   * hello=Hello
69   * cancel=Cancel
70   *
71   * #messages_de.properties
72   * hell=Hallo
73   * ok=OK
74   * </pre>
75   * <p>
76   * The Translation check will find the typo in the German {@code hello} key,
77   * the missing {@code ok} key in the default resource file and the missing
78   * {@code cancel} key in the German resource file:
79   * </p>
80   * <pre>
81   * messages_de.properties: Key 'hello' missing.
82   * messages_de.properties: Key 'cancel' missing.
83   * messages.properties: Key 'hell' missing.
84   * messages.properties: Key 'ok' missing.
85   * </pre>
86   * <p>
87   * Language code for the property {@code requiredTranslations} is composed of
88   * the lowercase, two-letter codes as defined by
89   * <a href="https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes">ISO 639-1</a>.
90   * Default value is empty String Set which means that only the existence of default
91   * translation is checked. Note, if you specify language codes (or just one
92   * language code) of required translations the check will also check for existence
93   * of default translation files in project.
94   * </p>
95   * <p>
96   * Attention: the check will perform the validation of ISO codes if the option
97   * is used. So, if you specify, for example, "mm" for language code,
98   * TranslationCheck will rise violation that the language code is incorrect.
99   * </p>
100  * <p>
101  * Attention: this Check could produce false-positives if it is used with
102  * <a href="https://checkstyle.org/config.html#Checker">Checker</a> that use cache
103  * (property "cacheFile") This is known design problem, will be addressed at
104  * <a href="https://github.com/checkstyle/checkstyle/issues/3539">issue</a>.
105  * </p>
106  * <ul>
107  * <li>
108  * Property {@code fileExtensions} - Specify file type extension to identify
109  * translation files. Setting this property is typically only required if your
110  * translation files are preprocessed and the original files do not have
111  * the extension {@code .properties}
112  * Type is {@code java.lang.String[]}.
113  * Default value is {@code .properties}.
114  * </li>
115  * <li>
116  * Property {@code baseName} - Specify
117  * <a href="https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/util/ResourceBundle.html">
118  * Base name</a> of resource bundles which contain message resources.
119  * It helps the check to distinguish config and localization resources.
120  * Type is {@code java.util.regex.Pattern}.
121  * Default value is {@code "^messages.*$"}.
122  * </li>
123  * <li>
124  * Property {@code requiredTranslations} - Specify language codes of required
125  * translations which must exist in project.
126  * Type is {@code java.lang.String[]}.
127  * Default value is {@code ""}.
128  * </li>
129  * </ul>
130  * <p>
131  * Note, that files with the same path and base name but which have different
132  * extensions will be considered as files that belong to different resource bundles.
133  * </p>
134  * <p>
135  * To configure the check to check only files which have '.properties' and
136  * '.translations' extensions:
137  * </p>
138  * <pre>
139  * &lt;module name="Translation"&gt;
140  *   &lt;property name="fileExtensions" value="properties, translations"/&gt;
141  *   &lt;property name="requiredTranslations" value="fr"/&gt;
142  * &lt;/module&gt;
143  * </pre>
144  * <p>
145  * Example:
146  * </p>
147  * <pre>
148  * #messages.properties
149  * hello=Hello
150  * cancel=Cancel
151  *
152  * #messages.translations
153  * hello=Hallo
154  * ok=OK
155  * </pre>
156  * <pre>
157  * messages.properties: Properties file 'messages_fr.properties' is missing.
158  * messages.translations: Properties file 'messages_fr.translations' is missing.
159  * </pre>
160  * <p>
161  * An example of how to configure the check to validate only bundles which base
162  * names start with "ButtonLabels":
163  * </p>
164  * <pre>
165  * &lt;module name="Translation"&gt;
166  *   &lt;property name="baseName" value="^ButtonLabels.*$"/&gt;
167  *   &lt;property name="requiredTranslations" value="fr"/&gt;
168  * &lt;/module&gt;
169  * </pre>
170  * <p>
171  * Example:
172  * </p>
173  * <pre>
174  * #ButtonLabels.properties
175  * hello=Hello
176  * cancel=Cancel
177  *
178  * #ButtonLabels_fr.properties
179  * hello=Bonjour
180  * name=Nom
181  * </pre>
182  * <pre>
183  * ButtonLabels.properties: Key 'name' is missing.
184  * ButtonLabels_fr.properties: Key 'cancel' is missing.
185  * </pre>
186  * <p>
187  * To configure the check to check existence of Japanese and French translations:
188  * </p>
189  * <pre>
190  * &lt;module name="Translation"&gt;
191  *   &lt;property name="requiredTranslations" value="ja, fr"/&gt;
192  * &lt;/module&gt;
193  * </pre>
194  * <p>
195  * Example:
196  * </p>
197  * <pre>
198  * #messages.properties
199  * hello=Hello
200  * cancel=Cancel
201  *
202  * #messages_ja.properties
203  * greeting=こんにちは
204  * age=年齢
205  *
206  * #messages_fr.properties
207  * greeting=Bonjour
208  * name=Nom
209  * </pre>
210  * <pre>
211  * messages.properties: Key 'age' missing.
212  * messages.properties: Key 'name' missing.
213  * messages_fr.properties: Key 'age' missing.
214  * messages_fr.properties: Key 'cancel' missing.
215  * messages_ja.properties: Key 'cancel' missing.
216  * messages_ja.properties: Key 'name' missing.
217  * </pre>
218  * <p>
219  * Parent is {@code com.puppycrawl.tools.checkstyle.Checker}
220  * </p>
221  * <p>
222  * Violation Message Keys:
223  * </p>
224  * <ul>
225  * <li>
226  * {@code translation.missingKey}
227  * </li>
228  * <li>
229  * {@code translation.missingTranslationFile}
230  * </li>
231  * </ul>
232  *
233  * @since 3.0
234  */
235 @GlobalStatefulCheck
236 public class TranslationCheck extends AbstractFileSetCheck {

347     /**
348      * Validates the correctness of user specified language codes for the check.
349      *
350      * @param languageCodes user specified language codes for the check.
351      * @throws IllegalArgumentException when any item of languageCodes is not valid language code
352      */
353     private void validateUserSpecifiedLanguageCodes(Set<String> languageCodes) {
354         for (String code : languageCodes) {
355             if (!isValidLanguageCode(code)) {
356                 final LocalizedMessage msg = new LocalizedMessage(TRANSLATION_BUNDLE,
357                         getClass(), WRONG_LANGUAGE_CODE_KEY, code);
358                 throw new IllegalArgumentException(msg.getMessage());
359             }
360         }
361     }

363     /**
364      * Checks whether user specified language code is correct (is contained in available locales).
365      *
366      * @param userSpecifiedLanguageCode user specified language code.
367      * @return true if user specified language code is correct.
368      */
369     private static boolean isValidLanguageCode(final String userSpecifiedLanguageCode) {
370         boolean valid = false;
371         final Locale[] locales = Locale.getAvailableLocales();
372         for (Locale locale : locales) {
373             if (userSpecifiedLanguageCode.equals(locale.toString())) {
374                 valid = true;
375                 break;
376             }
377         }
378         return valid;
379     }

392     @Override

394         final Set<ResourceBundle> bundles = groupFilesIntoBundles(filesToProcess, baseName);
395         for (ResourceBundle currentBundle : bundles) {
396             checkExistenceOfDefaultTranslation(currentBundle);
397             checkExistenceOfRequiredTranslations(currentBundle);

399         }

402     /**
403      * Checks an existence of default translation file in the resource bundle.
404      *
405      * @param bundle resource bundle.
406      */
407     private void checkExistenceOfDefaultTranslation(ResourceBundle bundle) {
408         getMissingFileName(bundle, null)
409             .ifPresent(fileName -> logMissingTranslation(bundle.getPath(), fileName));
410     }

412     /**
413      * Checks an existence of translation files in the resource bundle.
414      * The name of translation file begins with the base name of resource bundle which is followed
415      * by '_' and a language code (country and variant are optional), it ends with the extension
416      * suffix.
417      *
418      * @param bundle resource bundle.
419      */
420     private void checkExistenceOfRequiredTranslations(ResourceBundle bundle) {
421         for (String languageCode : requiredTranslations) {
422             getMissingFileName(bundle, languageCode)
423                 .ifPresent(fileName -> logMissingTranslation(bundle.getPath(), fileName));
424         }
425     }

427     /**
428      * Returns the name of translation file which is absent in resource bundle or Guava's Optional,
429      * if there is not missing translation.
430      *
431      * @param bundle resource bundle.
432      * @param languageCode language code.
433      * @return the name of translation file which is absent in resource bundle or Guava's Optional,
434      *         if there is not missing translation.
435      */
436     private static Optional<String> getMissingFileName(ResourceBundle bundle, String languageCode) {
437         final String fileNameRegexp;
438         final boolean searchForDefaultTranslation;
439         final String extension = bundle.getExtension();
440         final String baseName = bundle.getBaseName();
441         if (languageCode == null) {
442             searchForDefaultTranslation = true;
443             fileNameRegexp = String.format(Locale.ROOT, REGEXP_FORMAT_TO_CHECK_DEFAULT_TRANSLATIONS,
444                     baseName, extension);
445         }
446         else {
447             searchForDefaultTranslation = false;
448             fileNameRegexp = String.format(Locale.ROOT,
449                 REGEXP_FORMAT_TO_CHECK_REQUIRED_TRANSLATIONS, baseName, languageCode, extension);
450         }
451         Optional<String> missingFileName = Optional.empty();
452         if (!bundle.containsFile(fileNameRegexp)) {
453             if (searchForDefaultTranslation) {
454                 missingFileName = Optional.of(String.format(Locale.ROOT,
455                         DEFAULT_TRANSLATION_FILE_NAME_FORMATTER, baseName, extension));
456             }
457             else {
458                 missingFileName = Optional.of(String.format(Locale.ROOT,
459                         FILE_NAME_WITH_LANGUAGE_CODE_FORMATTER, baseName, languageCode, extension));
460             }
461         }
462         return missingFileName;
463     }

465     /**
466      * Logs that translation file is missing.
467      *
468      * @param filePath file path.
469      * @param fileName file name.
470      */
471     private void logMissingTranslation(String filePath, String fileName) {
472         final MessageDispatcher dispatcher = getMessageDispatcher();
473         dispatcher.fireFileStarted(filePath);
474         log(1, MSG_KEY_MISSING_TRANSLATION_FILE, fileName);
475         fireErrors(filePath);
476         dispatcher.fireFileFinished(filePath);
477     }

479     /**
480      * Groups a set of files into bundles.
481      * Only files, which names match base name regexp pattern will be grouped.
482      *
483      * @param files set of files.
484      * @param baseNameRegexp base name regexp pattern.
485      * @return set of ResourceBundles.
486      */
487     private static Set<ResourceBundle> groupFilesIntoBundles(Set<File> files,
488                                                              Pattern baseNameRegexp) {
489         final Set<ResourceBundle> resourceBundles = new HashSet<>();
490         for (File currentFile : files) {
491             final String fileName = currentFile.getName();
492             final String baseName = extractBaseName(fileName);
493             final Matcher baseNameMatcher = baseNameRegexp.matcher(baseName);
494             if (baseNameMatcher.matches()) {
495                 final String extension = CommonUtil.getFileExtension(fileName);
496                 final String path = getPath(currentFile.getAbsolutePath());
497                 final ResourceBundle newBundle = new ResourceBundle(baseName, path, extension);
498                 final Optional<ResourceBundle> bundle = findBundle(resourceBundles, newBundle);
499                 if (bundle.isPresent()) {
500                     bundle.get().addFile(currentFile);
501                 }
502                 else {
503                     newBundle.addFile(currentFile);
504                     resourceBundles.add(newBundle);
505                 }
506             }
507         }
508         return resourceBundles;
509     }

511     /**
512      * Searches for specific resource bundle in a set of resource bundles.
513      *
514      * @param bundles set of resource bundles.
515      * @param targetBundle target bundle to search for.
516      * @return Guava's Optional of resource bundle (present if target bundle is found).
517      */
518     private static Optional<ResourceBundle> findBundle(Set<ResourceBundle> bundles,
519                                                        ResourceBundle targetBundle) {
520         Optional<ResourceBundle> result = Optional.empty();
521         for (ResourceBundle currentBundle : bundles) {
522             if (targetBundle.getBaseName().equals(currentBundle.getBaseName())
523                     && targetBundle.getExtension().equals(currentBundle.getExtension())
524                     && targetBundle.getPath().equals(currentBundle.getPath())) {
525                 result = Optional.of(currentBundle);
526                 break;
527             }
528         }
529         return result;
530     }

532     /**
533      * Extracts the base name (the unique prefix) of resource bundle from translation file name.
534      * For example "messages" is the base name of "messages.properties",
535      * "messages_de_AT.properties", "messages_en.properties", etc.
536      *
537      * @param fileName the fully qualified name of the translation file.
538      * @return the extracted base name.
539      */
540     private static String extractBaseName(String fileName) {
541         final String regexp;
542         final Matcher languageCountryVariantMatcher =
543             LANGUAGE_COUNTRY_VARIANT_PATTERN.matcher(fileName);
544         final Matcher languageCountryMatcher = LANGUAGE_COUNTRY_PATTERN.matcher(fileName);
545         final Matcher languageMatcher = LANGUAGE_PATTERN.matcher(fileName);
546         if (languageCountryVariantMatcher.matches()) {
547             regexp = LANGUAGE_COUNTRY_VARIANT_PATTERN.pattern();
548         }
549         else if (languageCountryMatcher.matches()) {
550             regexp = LANGUAGE_COUNTRY_PATTERN.pattern();
551         }
552         else if (languageMatcher.matches()) {
553             regexp = LANGUAGE_PATTERN.pattern();
554         }
555         else {
556             regexp = DEFAULT_TRANSLATION_REGEXP;
557         }
558         // We use substring(...) instead of replace(...), so that the regular expression does
559         // not have to be compiled each time it is used inside 'replace' method.
560         final String removePattern = regexp.substring("^.+".length());
561         return fileName.replaceAll(removePattern, "");
562     }

564     /**
565      * Extracts path from a file name which contains the path.
566      * For example, if the file name is /xyz/messages.properties,
567      * then the method will return /xyz/.
568      *
569      * @param fileNameWithPath file name which contains the path.
570      * @return file path.
571      */
572     private static String getPath(String fileNameWithPath) {
573         return fileNameWithPath
574             .substring(0, fileNameWithPath.lastIndexOf(File.separator));
575     }

577     /**
578      * Checks resource files in bundle for consistency regarding their keys.
579      * All files in bundle must have the same key set. If this is not the case
580      * an audit event message is posted giving information which key misses in which file.
581      *
582      * @param bundle resource bundle.
583      */
584     private void checkTranslationKeys(ResourceBundle bundle) {
585         final Set<File> filesInBundle = bundle.getFiles();
586         // build a map from files to the keys they contain
587         final Set<String> allTranslationKeys = new HashSet<>();
588         final Map<File, Set<String>> filesAssociatedWithKeys = new TreeMap<>();
589         for (File currentFile : filesInBundle) {
590             final Set<String> keysInCurrentFile = getTranslationKeys(currentFile);
591             allTranslationKeys.addAll(keysInCurrentFile);
592             filesAssociatedWithKeys.put(currentFile, keysInCurrentFile);
593         }
594         checkFilesForConsistencyRegardingTheirKeys(filesAssociatedWithKeys, allTranslationKeys);
595     }
596 
597     /**
598      * Compares th the specified key set with the key sets of the given translation files (arranged
599      * in a map). All missing keys are reported.
600      *
601      * @param fileKeys a Map from translation files to their key sets.
602      * @param keysThatMustExist the set of keys to compare with.
603      */
604     private void checkFilesForConsistencyRegardingTheirKeys(Map<File, Set<String>> fileKeys,
605                                                             Set<String> keysThatMustExist) {
606         for (Entry<File, Set<String>> fileKey : fileKeys.entrySet()) {
607             final Set<String> currentFileKeys = fileKey.getValue();
608             final Set<String> missingKeys = keysThatMustExist.stream()
609                 .filter(key -> !currentFileKeys.contains(key)).collect(Collectors.toSet());
610             if (!missingKeys.isEmpty()) {
611                 final MessageDispatcher dispatcher = getMessageDispatcher();
612                 final String path = fileKey.getKey().getAbsolutePath();
613                 dispatcher.fireFileStarted(path);
614                 for (Object key : missingKeys) {
615                     log(1, MSG_KEY, key);
616                 }
617                 fireErrors(path);
618                 dispatcher.fireFileFinished(path);
619             }
620         }
621     }

623     /**
624      * Loads the keys from the specified translation file into a set.
625      *
626      * @param file translation file.
627      * @return a Set object which holds the loaded keys.
628      */
629     private Set<String> getTranslationKeys(File file) {
630         Set<String> keys = new HashSet<>();
631         try (InputStream inStream = Files.newInputStream(file.toPath())) {
632             final Properties translations = new Properties();
633             translations.load(inStream);
634             keys = translations.stringPropertyNames();
635         }
636         // -@cs[IllegalCatch] It is better to catch all exceptions since it can throw
637         // a runtime exception.
638         catch (final Exception ex) {
639             logException(ex, file);
640         }
641         return keys;
642     }

645      * Helper method to log an exception.
646      *
647      * @param exception the exception that occurred
648      * @param file the file that could not be processed
649      */
650     private void logException(Exception exception, File file) {
651         final String[] args;
652         final String key;
653         if (exception instanceof NoSuchFileException) {
654             args = null;
655             key = "general.fileNotFound";
656         }
657         else {
658             args = new String[] {exception.getMessage()};
659             key = "general.exception";
660         }
661         final Violation message =
662             new Violation(
663                 0,
664                 Definitions.CHECKSTYLE_BUNDLE,
665                 key,
666                 args,
667                 getId(),
668                 getClass(), null);
669         final SortedSet<Violation> messages = new TreeSet<>();
670         messages.add(message);
671         getMessageDispatcher().fireErrors(file.getPath(), messages);
672         log.debug("Exception occurred.", exception);
673     }

675     /** Class which represents a resource bundle. */
676     private static final class ResourceBundle {
677 
678         /** Bundle base name. */
679         private final String baseName;
680         /** Common extension of files which are included in the resource bundle. */
681         private final String extension;
682         /** Common path of files which are included in the resource bundle. */
683         private final String path;
684         /** Set of files which are included in the resource bundle. */
685         private final Set<File> files;
686 
687         /**
688          * Creates a ResourceBundle object with specific base name, common files extension.
689          *
690          * @param baseName bundle base name.
691          * @param path common path of files which are included in the resource bundle.
692          * @param extension common extension of files which are included in the resource bundle.
693          */
694         private ResourceBundle(String baseName, String path, String extension) {
695             this.baseName = baseName;
696             this.path = path;
697             this.extension = extension;
698             files = new HashSet<>();
699         }
700 
701         /**
702          * Returns the bundle base name.
703          *
704          * @return the bundle base name
705          */
706         public String getBaseName() {
707             return baseName;
708         }
709 
710         /**
711          * Returns the common path of files which are included in the resource bundle.
712          *
713          * @return the common path of files
714          */
715         public String getPath() {
716             return path;
717         }
718 
719         /**
720          * Returns the common extension of files which are included in the resource bundle.
721          *
722          * @return the common extension of files
723          */
724         public String getExtension() {
725             return extension;
726         }
727 
728         /**
729          * Returns the set of files which are included in the resource bundle.
730          *
731          * @return the set of files
732          */
733         public Set<File> getFiles() {
734             return Collections.unmodifiableSet(files);
735         }
736 
737         /**
738          * Adds a file into resource bundle.
739          *
740          * @param file file which should be added into resource bundle.
741          */
742         public void addFile(File file) {
743             files.add(file);
744         }
745 
746         /**
747          * Checks whether a resource bundle contains a file which name matches file name regexp.
748          *
749          * @param fileNameRegexp file name regexp.
750          * @return true if a resource bundle contains a file which name matches file name regexp.
751          */
752         public boolean containsFile(String fileNameRegexp) {
753             boolean containsFile = false;
754             for (File currentFile : files) {
755                 if (Pattern.matches(fileNameRegexp, currentFile.getName())) {
756                     containsFile = true;
757                     break;
758                 }
759             }
760             return containsFile;
761         }

763     }
